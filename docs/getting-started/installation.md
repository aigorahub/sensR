# Installation

## Requirements

- Python 3.10 or higher
- NumPy, SciPy, Pandas
- Plotly (for interactive visualizations)

## Install from PyPI

```bash
pip install senspy
```

## Install from Source

```bash
git clone https://github.com/aigorahub/sensPy.git
cd sensPy
pip install -e .
```

## Using uv (recommended)

```bash
uv pip install senspy
```

## Verify Installation

```python
import senspy
print(senspy.__version__)
```

## Optional Dependencies

For development:

```bash
pip install senspy[dev]
```
